# Carlos Alberto Aguilar Lazcano Portfolio

Static personal portfolio website for Carlos Alberto Aguilar Lazcano, built with plain HTML, CSS, and vanilla JavaScript.

## Files

- `index.html` - Page structure and content.
- `styles.css` - Responsive visual design.
- `script.js` - Mobile navigation, active section state, and dynamic year.
- `README.md` - Publishing notes.

## Run Locally

Open `index.html` directly in a browser. No build step, package manager, backend, or framework is required.

## Publish With GitHub Pages

1. Create a GitHub repository for the portfolio.
2. Add `index.html`, `styles.css`, `script.js`, and `README.md` to the repository root.
3. Commit and push the files to GitHub.
4. In the repository, open **Settings**.
5. Go to **Pages**.
6. Under **Build and deployment**, choose **Deploy from a branch**.
7. Select the branch that contains the site, usually `main`, and choose `/root`.
8. Save the settings.

GitHub Pages will provide a public URL after the first deployment finishes.
